# SISTEMA_INVENTARIO
Sistema de Inventario para el curso Técnicas POO

## Descripción del Proyecto
Este proyecto es un sistema de inventario desarrollado como parte del curso de Técnicas de Programación Orientada a Objetos (POO). El sistema permite gestionar productos, categorías y proveedores de manera eficiente.

## Funcionalidades Principales
- Registro y gestión de productos
- Categorización de productos
- Gestión de proveedores
- Consultas y reportes del inventario

## Tecnologías Utilizadas
- Java 21
- Spring Boot
- MySQL
- Maven

## Requisitos del Sistema
- Java Development Kit (JDK) 17 o superior
- Apache Maven 3.6 o superior
- MySQL 8.0 o superior

## Instrucciones de Instalación
1. Clonar el repositorio
2. Configurar la base de datos MySQL
3. Actualizar las credenciales de conexión en `application.properties`
4. Ejecutar el proyecto con Maven

## Instrucciones de Uso
1. Iniciar el servidor Spring Boot
2. Acceder a la aplicación en `http://localhost:8080`
3. Utilizar la interfaz gráfica para gestionar el inventario

## Arquitectura del Sistema
El sistema sigue los principios de programación orientada a objetos con una arquitectura modular que incluye:
- Capa de modelo (entidades)
- Capa de servicio (lógica de negocio)
- Capa de controlador (controladores REST)
- Capa de acceso a datos (repositorios)
